# Landing Page Redesign Preview

## Design Direction: Brutalist-Inspired SaaS

### Typography Stack
- **Headlines**: "Newsreader", "Fraunces" (editorial serif)
- **Body**: "Space Grotesk", "DM Sans" (geometric sans, tight spacing)
- **Accents**: "Azeret Mono" (technical mono for prices/data)

### Layout Philosophy
- Asymmetric grids (2/3 + 1/3 splits instead of centered)
- Offset text blocks that break the grid
- Overlapping cards with sharp shadows
- Full-bleed sections alternating with contained content
- No rounded corners on primary elements (sharp rectangles)

### Color Strategy
- Monochrome base (true black #000 / true white #FFF)
- Single accent: Electric blue (#0066FF) used sparingly
- Heavy borders (2-3px) instead of subtle hairlines
- No gradients - flat colors only

### Unique Elements
- Offset heading blocks that hang outside containers
- Price displayed in large mono numerals (96px+)
- Feature cards as newspaper-style columns
- Comparison table with zebra stripes (bold)
- Footer as dense information block (no whitespace luxury)
